安装指南

几分钟内开始使用 AI Components 组件库。

📦 安装

npm / pnpm / yarn
# 选择你的包管理器
npm install ai-components
pnpm add ai-components
yarn add ai-components
CDN（<script> 标签）
<script src="https://unpkg.com/ai-components/dist/ai-components.iife.global.js"></script>
💡 提示：IIFE 产物加载后会自动注册所有 AI 组件（ai-component、ai-text、ai-markdown 等）， 并暴露 AIC 全局变量。你只需调用 AIC.configure() 配置 LLM 请求函数即可使用。

⚡ 快速开始

推荐 方式一：ESM 模块 

import { configure, defineAIComponent } from 'ai-components';

// 1. 提供 LLM 请求函数（必须返回 AsyncIterable<string>）
configure({
  request: async function*(prompt, options) {
    // 调用你的 LLM API，逐 chunk yield 文本
    const res = await fetch('/api/chat', {
      method: 'POST',
      body: JSON.stringify({ prompt }),
      signal: options?.signal,
    });
    const reader = res.body.getReader();
    // ...解析 SSE 并 yield content
  }
});

// 2. 注册所有 AI 组件
defineAIComponent();

方式二：<script> 标签

<script src="ai-components/dist/ai-components.iife.global.js"></script>
<script>
  AIC.configure({ apiKey: 'sk-...' });
</script>

然后在 HTML 中使用

<!-- 设置 p 属性，组件自动调用 LLM 并流式渲染 -->
<ai-component p="生成一个精美的用户注册表单"></ai-component>

<!-- 组件之间可以嵌套！LLM 输出的 HTML 中如果包含 <ai-component>，
     它们也会自动注册和渲染 -->
<ai-component p="生成一个仪表盘布局，其中包含多个子 ai-component 分别展示不同数据卡片"></ai-component>

🔧 配置 LLM

configure() 是使用任何 AI 组件前的必要步骤。 最简单的方式是只传一个 apiKey，组件库内置了 OpenAI 兼容的流式请求逻辑。

简化配置（推荐）

// 只需 API Key，默认使用 DeepSeek API
configure({ apiKey: 'sk-...' })

// 可选：指定 baseUrl 和 model 切换提供商
configure({
  apiKey: 'sk-...',
  baseUrl: 'https://api.openai.com/v1',
  model: 'gpt-4o',
})

高级：自定义请求函数

如果你需要完全控制请求逻辑（如自定义鉴权、代理、非 OpenAI 格式的 API），可以提供一个 request 函数，它接收 prompt 和 options，返回 AsyncIterable<string>： 

configure({
  request: async function*(prompt, options) {
    // 调用你的 API，逐 chunk yield 文本内容
    const res = await fetch('/api/chat', { ... });
    // ... 解析响应并 yield 每个文本片段
    yield 'hello';
    yield ' world';
  }
})

🎯 极简接入示例

只需一个 HTML 文件，引入脚本 + 一行配置，即可使用所有 AI 组件。复制下面的代码保存为 index.html，用浏览器打开即可运行。 

<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8" />
  <title>AI Components 极简示例</title>
  <script src="https://cdn.tailwindcss.com"></script>
  <script src="https://unpkg.com/ai-components/dist/ai-components.iife.global.js"></script>
</head>
<body>

<ai-component p="生成一个精美的用户注册表单"></ai-component>

<script>
  AIC.configure({ apiKey: 'YOUR_API_KEY' })
</script>
</body>
</html>
⚠️ 注意：将 YOUR_API_KEY 替换为你的真实 API Key。 默认使用 DeepSeek API（deepseek-chat 模型）。
💡 就这么简单！整个页面只需要：
  1. 引入 IIFE 脚本（自动注册所有组件）
  2. 调用 AIC.configure({ apiKey })
  3. 在 HTML 中放 AI 组件标签

切换 API 提供商

通过 baseUrl 和 model 参数切换到其他 OpenAI 兼容 API：

// DeepSeek（默认）
AIC.configure({ apiKey: 'sk-...' })

// OpenAI
AIC.configure({ apiKey: 'sk-...', baseUrl: 'https://api.openai.com/v1', model: 'gpt-4o' })

// Gemini
AIC.configure({ apiKey: '...', baseUrl: 'https://generativelanguage.googleapis.com/v1beta/openai', model: 'gemini-2.5-flash' })

// 通义千问
AIC.configure({ apiKey: 'sk-...', baseUrl: 'https://dashscope.aliyuncs.com/compatible-mode/v1', model: 'qwen-plus' })

更多组件示例

同一个页面中可以使用所有 AI 组件，它们都已自动注册：

<!-- AI 生成 HTML 页面 -->
<ai-component p="生成一个天气预报卡片"></ai-component>

<!-- AI 生成纯文本 -->
<ai-text p="写一句励志名言" tag="h2"></ai-text>

<!-- AI 生成 Markdown 并渲染 -->
<ai-markdown p="写一篇 JavaScript 闭包教程"></ai-markdown>

<!-- AI 搜索图片 -->
<ai-image p="夕阳下的海边灯塔" width="600"></ai-image>

<!-- AI 地图定位 -->
<ai-map p="北京天安门"></ai-map>

<!-- AI 音乐播放 -->
<ai-music p="播放一首轻松的爵士乐"></ai-music>

<!-- 数据驱动：先获取数据，再渲染 UI -->
<ai-component p="渲染一个购物车列表">
  <ai-data p="mock 5 件商品数据" />
</ai-component>
