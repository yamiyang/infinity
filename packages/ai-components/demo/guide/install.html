安装指南

几分钟内开始使用 AI Components 组件库。

📦 安装

npm / pnpm / yarn
# 选择你的包管理器
npm install ai-components
pnpm add ai-components
yarn add ai-components
CDN（<script> 标签）
<script src="https://unpkg.com/ai-components/dist/ai-components.iife.global.js"></script>
💡 提示：IIFE 产物会自动注册 AIC 全局变量并调用 defineAIComponent()，你只需 AIC.configure() 即可使用。

⚡ 快速开始

推荐 方式一：ESM 模块 

import { configure, defineAIComponent } from 'ai-components';

// 1. 提供 LLM 请求函数（必须返回 AsyncIterable<string>）
configure({
  request: async function*(prompt, options) {
    // 调用你的 LLM API，逐 chunk yield 文本
    const res = await fetch('/api/chat', {
      method: 'POST',
      body: JSON.stringify({ prompt }),
      signal: options?.signal,
    });
    const reader = res.body.getReader();
    // ...解析 SSE 并 yield content
  }
});

// 2. 注册所有 AI 组件
defineAIComponent();

方式二：<script> 标签

<script src="ai-components/dist/ai-components.iife.global.js"></script>
<script>
  AIC.configure({ request: yourRequestFn });
</script>

然后在 HTML 中使用

<!-- 设置 p 属性，组件自动调用 LLM 并流式渲染 -->
<ai-component p="生成一个精美的用户注册表单"></ai-component>

<!-- 组件之间可以嵌套！LLM 输出的 HTML 中如果包含 <ai-component>，
     它们也会自动注册和渲染 -->
<ai-component p="生成一个仪表盘布局，其中包含多个子 ai-component 分别展示不同数据卡片"></ai-component>

🔧 配置 LLM

configure() 是使用任何 AI 组件前的必要步骤。你需要提供一个 LLMRequestFn，它接收 prompt 和 options，返回 AsyncIterable<string>。 

// 示例：接入 OpenAI 兼容 API
async function* openaiRequest(prompt, options) {
  const res = await fetch('https://api.openai.com/v1/chat/completions', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': `Bearer ${API_KEY}`,
    },
    body: JSON.stringify({
      model: 'gpt-4o',
      messages: [
        { role: 'system', content: options?.systemPrompt || '' },
        { role: 'user', content: prompt },
      ],
      stream: true,
    }),
    signal: options?.signal,
  });

  // 解析 SSE 流...
  const reader = res.body.getReader();
  // ... yield 每个 delta.content
}

configure({ request: openaiRequest });
