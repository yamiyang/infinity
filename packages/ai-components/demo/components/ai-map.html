<ai-map>

可用

AI 驱动的智能地图组件——通过自然语言描述地点、路线或区域，自动解析位置信息并展示交互式地图。 基于 OpenStreetMap，无需 API Key 即可使用。如果配置了 LLM，会使用 AI 更精准地理解位置描述。

📖 文档
🎮 Playground

⚙️ 属性（Attributes）

属性类型默认值说明
p string — 必填。自然语言描述位置、地点或路线。修改后重新加载。
width number 600 地图宽度（像素）
height number 400 地图高度（像素）
zoom number 14 缩放级别（3=世界, 10=城市, 14=街区, 18=建筑）

💡 使用示例

基本用法 — 单个地点

<ai-map p="北京天安门"></ai-map>

指定尺寸和缩放

<ai-map
  p="东京塔"
  width="800"
  height="500"
  zoom="16"
></ai-map>

英文地址

<ai-map p="Times Square, New York" zoom="15"></ai-map>

🗺️ 工作原理

1. 解析位置（双重策略）
策略 A：LLM 理解（需配置 LLM）— 使用 AI 将自然语言转为结构化坐标（纬度、经度、缩放级别）。
策略 B：地理编码（免费兜底）— 使用 OpenStreetMap Nominatim 将地名转为坐标。
2. 渲染地图
使用 OpenStreetMap 嵌入式地图展示位置，包含标记点和信息栏。完全免费，无需注册。
3. 交互功能
顶部显示位置名称和坐标信息，底部提供"在 OpenStreetMap 中查看"的链接。
